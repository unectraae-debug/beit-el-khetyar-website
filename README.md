# Beit El Khetyar Website

Premium restaurant website built from the provided Beit El Khetyar menu design.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Assets

- Logo placeholder extracted from the menu: `public/images/logo.png`
- Menu PDF: `public/menu.pdf`
- Exported menu pages: `public/images/menu/page-1.jpg` through `page-8.jpg`

## Main features

- Cream, olive, sage, and charcoal styling based on the menu.
- Sticky header with logo on top left.
- Bilingual Arabic/English content.
- Interactive two-page flipbook menu using `@pageflip/react-pageflip`.
- Zoom, fullscreen, previous/next, PDF download controls.
- Branch cards for Al Najda, Hamdan, and Baniyas.
